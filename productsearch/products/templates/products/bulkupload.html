Upload a CSV file to add products to Database in bulk

  • The file must NOT include a header row
  • Each row of the file must have the following form:
  • ID,description,lastSold,shelfLife,department,price,unit,xFor,cost
  • lastSold must be a date with the form MM/DD/YYYY
  • shelfLife must be in the form Nd where N is an integer
  • department must be either Produce, Grocery, or Pharmacy
  • unit must be either lb or each
  • price and cost must have the form $X.XX
  • xFor must be an integer
